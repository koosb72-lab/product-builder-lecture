
# Lotto Number Generator

## Overview

This is a simple web application that generates and displays a set of 6 unique lottery numbers between 1 and 45.

## Design and Features

### Visual Design

*   **Layout:** Centered content with a clean and modern look.
*   **Colors:** A color-coded system for the lottery numbers to easily distinguish between ranges (1-10, 11-20, etc.).
    *   1-10: Yellow
    *   11-20: Blue
    *   21-30: Red
    *   31-40: Gray
    *   41-45: Green
*   **Typography:** System fonts for a native feel.
*   **Interactivity:** A "Generate Numbers" button with a hover effect.

### Features

*   **Number Generation:** Generates 6 unique random numbers from 1 to 45.
*   **Display:** Displays the generated numbers in colored circles.
*   **User Interaction:** A button to generate a new set of numbers.
